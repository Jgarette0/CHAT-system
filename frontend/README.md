# React Template

Template for a React project with the following features:

- **Tailwind CSS** : For css
- **Shadcn UI**: Shadcn UI for UI components.
- **React Router**: Router for routing
- **Dark Mode Setup**: Dark mode theme.
- **Public Font & Image**.
